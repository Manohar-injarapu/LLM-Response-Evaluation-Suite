# LLM-Response-Evaluation-Suite
Testing an AI customer support chatbot for an Indian e-commerce platform

This project implements a **22-test case evaluation suite** for an LLM-based customer support chatbot, covering:

-  Prompt Injection Testing (5 attack vectors)
-  Hallucination Detection (5 knowledge-boundary scenarios)
-  Bias & Fairness Validation (5 paired comparison tests)
-  Response Consistency Testing (5 rephrasing scenarios)

---

## Key Achievements

- Identified **2 Critical Prompt Injection Vulnerabilities**
- Improved **AI system security and robustness**
- Enabled structured validation of **non-deterministic LLM outputs**
